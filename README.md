AlpCore
=======

Core requirement for Alpvax's Minecraft mods. More of a library than an actual mod.<br>

Ensure USER_HOME eclipse environment variable is set in order for classpath to work.<br>

To update forge version, update the build.gradle file then run updateClasspath.bat<br>
Copy post-merge to .git/hooks to automatically re-setup the workspace when build.gradle has changed in the remote
